# Resume vs Job Description Matcher

## Overview
This project compares a candidate’s resume with a job description and evaluates how well they match.  
It identifies overlapping skills, missing requirements, and generates a match score along with a recruiter-style summary.

The goal is to simulate an **ATS (Applicant Tracking System)** style screening process using Python.


## Features
- Resume and Job Description comparison
- Skill matching and gap analysis
- Match score calculation (0–100)
- Recruiter-style summary output
- Lightweight and beginner-friendly


## Tech Stack
- Python 3


## How It Works
1. Resume and Job Description text are provided as input
2. Relevant skills are extracted and compared
3. A match score is calculated based on skill overlap
4. Missing skills and suggestions are generated


## How to Run
```bash
python app.py
```
